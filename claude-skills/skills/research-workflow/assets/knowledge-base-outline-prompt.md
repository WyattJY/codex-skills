# 基于知识库的大纲撰写（Prompt 模板）

## 任务

请基于 Source 中与 **{{TOPIC}}** 相关的全部文献，构建一份深度综述报告大纲。

## 格式与内容要求

### 第 1 章：背景与动机（需完整撰写）

结合 Source 详细论述

### 第 2 章：技术演进路径与文献分类（仅列大纲，需细化至三级小节号）

**组织逻辑：** 二级标题为技术分类，按 Source 实际内容调整）。

**过渡要求：** 在每个二级标题下方，请撰写一段简短的过渡性文字，介绍该分类的技术逻辑、演进脉络及所选代表性论文的关联。

**三级标题格式：** 必须为单篇具体论文，且具备小节号。格式为：`2.x.y [年份] 论文标题/简称`。

**正文占位：** 三级标题下统一使用占位符：`[基于本篇论文的内容待后续详细生成]`。

### 第 3 章：趋势分析（需完整撰写）

### 第 4 章：可行性与局限性探讨（需完整撰写）

深入分析 **{{TOPIC}}** 落地可行性及目前存在的痛点。

## 整体要求

- **语言风格：** 学术严谨，具备深度技术洞察。
- **逻辑：** 体现文献之间的代际关系（Inheritance and Improvement）。
- **广度：** 确保不遗漏 Source 中的核心论文。
