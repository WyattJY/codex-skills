# NWAFU Master Thesis Writing Guide (Practical)

## Abstract (摘要) pattern

Write 1 paragraph (background → gap → your method) + 3–5 numbered contributions + 1 sentence of significance.
End with 3–6 keywords (关键词), ordered by importance.

## Literature review (文献综述) pattern

Use “evolution → limitation → motivation”:
1. 灌溉决策模型（从定量到动态）
2. 机器学习在灌溉决策中的应用
3. 多模态（传感器×图像表型）在灌溉中的应用
4. 智能体系统（把模型落地为可用系统；RAG/记忆/人机协同）

## Methods / system chapters

- State inputs/outputs clearly (symbols + units).
- Describe datasets before models.
- For each model/system module: “goal → design → implementation → evaluation”.
- Close each chapter with “本章小结” (what was done + what it enables next).

## Language & style checklist

- Use “本文/本研究/本章” instead of “我/我们”.
- Prefer concrete numbers (metrics, ablations, savings) over vague adjectives.
- Introduce abbreviations once (中文全称 + 英文缩写), then use the abbreviation consistently.
- Keep citations close to claims; avoid stacking citations at paragraph end.
