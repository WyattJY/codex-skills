# Hermes Local Skills

This directory mirrors the non-hidden contents of the local Hermes skill library from:

`/Users/jiangyu/.hermes/skills`

It is intended as a source backup for the skills Hermes can see locally. Runtime state, curator backups, usage files, hidden files, AppleDouble metadata, Python bytecode, and logs are excluded.

Notable included groups include:

- `autonomous-ai-agents`
- `software-development`
- `research`
- `github`
- `productivity`
- `creative`
- `media`
- `mlops`
- `apple`

Do not store API keys, OAuth tokens, cookies, `.env` files, or private company data in this directory.
